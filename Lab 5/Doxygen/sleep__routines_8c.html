◆ current_block_energy_mode()

uint32_t current_block_energy_mode ( void  )

Blocks the system from surpassing EM3.

Uses while loop to make sure the array of "lowest_energy_mode[]" doesn't surpass the 4th spot (EM3)

Note
Function breaks if EM4 is entered. This function must also handle interrupts with CORE_DECLARE_IRQ_STATE,CORE_ENTER_CRITICAL(),and CORE_EXIT_CRITICAL() which are described earlier
Returns
k returns the value of MAX_ENERGY_MODES-1, the energy mode the system cannot enter

◆ enter_sleep()

void enter_sleep ( void  )

Function to enter sleep.

CORE_DECLARE_IRQ_STATE - saves the current state of the global interrupt enable bit CORE_ENTER_CRITICAL()- Disables all global interrupts CORE_EXIT_CRITICAL()- If the saved value is zero, this will enable global interrupts but if the saved value is 1, global interrupts will remain disabled

Note
This lowest energy mode the system can enter is EM3
